PagePerfect Help Documentation

Welcome to the official help documentation for the PagePerfect™ HTML website editor from the PagePerfect™ Group! Here you can learn all about how to use the editor, cool tips, and behind-the-scenes information! So, let’s get started.

Using the Editor

PagePerfect™ is a very simple and easy-to-use WYSIWYG (“what you see is what you get”) HTML editor for the web. With it, you can create powerful webpages for any purpose, from articles, to homepages, to anything!

Buttons

You an interface with your PagePerfect™ projects using the main buttons available right below the app header.

Save Project

This button packages the editor’s current contents into a PagePerfect Project (.p3) file.

Open Project

This button allows you to select and open one of any available .p3 files on your system.

PagePerfect™ only allows you to open .p3 files to protect your projects and to prevent the editor from crashing. Random HTML files may break something—which isn’t good!

Download HTML

This button allows you to actually save the HTML code so you can utilize it on your websites.

Copy Page Code

This button allows you to copy the HTML code to your clipboard rather than having to save an HTML file.

Clear Page

This wipes the editor clean, removing everything so you can start again.

Get Help & More

This button opens the PagePerfect™ help documentation.
